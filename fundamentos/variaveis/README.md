# Variáveis em Java

Variáveis são utilizadas para armazenar dados na memória durante a execução do programa.
Em Java, toda variável possui um **tipo**, que define o tipo de valor que ela pode armazenar.

## Tipos de variáveis mais comuns
- `int` → números inteiros
- `double` → números decimais
- `String` → textos
- `boolean` → verdadeiro ou falso

Nesta pasta você encontrará exemplos simples para entender como declarar e usar variáveis em Java.
